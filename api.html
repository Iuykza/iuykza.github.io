users

Gets all users' data.

Usage

http://shiftswap.com/api/userid/:userid

Parameters

:userid
Integer. A user's unique 4 digit ID number.

Example

Send http://shiftswap.com/api/userid/1007

Receive {"userid":1007,"name":"Douglas S. Morrison","role":"gm","lastlog":"null"}

schedule/userid

Gets a work week for a user with all of its shift times and descriptions.

Usage

http://shiftswap.com/api/schedule/userid/:userid/:day

Parameters

:userid
Integer. A user's unique 4 digit ID number.
:day (optional)
String. UNIX timestamp. Any day of a particular work week. The week's Thursday and seven days afterwards will be returned. If blank, the current week is returned.

Example

Send http://shiftswap.com/api/schedule/userid/1007/1459486800

Receive

schedule/role

Gets a work week for all employees or managers with all of its shift times and descriptions.

Usage

http://shiftswap.com/api/schedule/role/:role/:day

Parameters

:role
String. Changes between the employee or manager schedule. Accepts: employee or manager.
:day (optional)
String. UNIX timestamp. Any day of a particular work week. The week's Thursday and seven days afterwards will be returned. If blank, the current week is returned.

Example

Send http://shiftswap.com/api/schedule/role/manager/1459486800

Receive
